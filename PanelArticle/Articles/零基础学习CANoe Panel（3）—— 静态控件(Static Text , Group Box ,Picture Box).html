📙 静态文本(Static Text )

  • 1️⃣ 静态文本,作为一个文本控件,可以用作标题,控件注释等作用.

  • 2️⃣ 字体和段落属性设置

  • 后面的大多数控件都会有Font 的属性设置，这里都尝试下，对后面的控件美感的设计是有帮助的。

  • 可以设置字体大小，加粗，斜体，颜色，背景色，剧中对齐等

  • 后面的所有控件，Font 属性将不会再做讲述。



📙 分组盒子(Group Box)：

1️⃣ Group Box 也是一个静态控件，不需要绑定任何的symbol,相当于一个容器，可以把把你想归类的控件都放在一起,让你的panel在布局和归类上更加清晰

2️⃣ Vector 官方的很多实例中的panel都会有Group Box的使用,它可以让我们的Panel功能分类更加清晰,随便截图两个看下

📙 图片盒子(Picture Box )

1️⃣ 图像盒子，和Group Box 一样，静态控件，容器，可以放置一张图片，用来做解释说明啥的，

2️⃣ 比如官方示例中模拟干扰的panel，贴一张图片就很好,可以很直观的感受到模拟干扰的硬件电路图
